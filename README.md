# Shivaditya Infotech Redesign

A modern React.js website redesign for Shivaditya Infotech using Vite, Tailwind CSS, React Router, and Framer Motion.

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React

## Project Structure

```text
src/
  components/
    layout/
    sections/
    shared/
  data/
  pages/
  App.jsx
  main.jsx
  index.css
```

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Brand colors are centralized in `tailwind.config.js` and `src/index.css`.
- Replace placeholder project, testimonial, blog, and careers content in `src/data/siteData.js`.
- The contact form currently demonstrates frontend interaction only. Connect it to your preferred backend or form service for production use.
